Installing Anaconda Python and Creating a Python Environment

1. Installing Anaconda Python

On Windows

  1. Download the Anaconda installer from the Anaconda website.
  2. Run the downloaded installer (.exe file).
  3. Follow the prompts in the installation wizard. Choose "Add Anaconda to my PATH environment variable" and click "Install".
  4. After the installation is complete, open the Anaconda Prompt from the Start Menu.

On macOS

  1. Download the Anaconda installer from the Anaconda website.
  2. Open the Terminal and navigate to the directory where the installer was downloaded.
  3. Run the installer with the following command: 
    bash Anaconda3-2024.05-MacOSX-x86_64.sh
  4. Follow the prompts in the Terminal to complete the installation.
  5. Close and reopen the Terminal, or run `source ~/.bash_profile` to activate the Anaconda environment.

On Ubuntu

  1. Download the Anaconda installer from the Anaconda website.
  2. Open the Terminal and navigate to the directory where the installer was downloaded.
  3. Run the installer with the following command: 
    bash Anaconda3-2024.05-Linux-x86_64.sh
  4. Follow the prompts in the Terminal to complete the installation.
  5. Close and reopen the Terminal, or run `source ~/.bashrc` to activate the Anaconda environment.

2. Creating a Python Environment

  1. Open your terminal (or Anaconda Prompt on Windows).
  2. Create a new environment with a specific Python version by running: 
    conda create --name myenv python=3.8
    Replace `myenv` with your desired environment name and `3.8` with your desired Python version.
  3. Activate the newly created environment: 
    conda activate myenv
  4. To install packages in this environment, use: 
    conda install package_name
    Replace `package_name` with the name of the package you want to install.
  5. To deactivate the environment, simply run: 
    conda deactivate
